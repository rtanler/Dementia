# Getting Started

## How to Use This Textbook

### Navigation

Use the left sidebar to navigate between different sections of the textbook:

- **Chapters**: Main content organized by topic
- **MicroSims**: Interactive simulations and visualizations
- **Learning Graph**: Visual representation of concept dependencies
- **Glossary**: Definitions of key terms
- **FAQ**: Frequently asked questions

### Features

#### Search
Use the search bar at the top to find specific topics or terms.

#### Code Blocks
Code examples include a copy button for easy copying.

#### Interactive Elements
Look for interactive MicroSims embedded throughout the chapters for hands-on learning.

#### Math Support
Mathematical expressions are rendered using MathJax. Inline math is denoted with `$...$` and block math with `$$...$$`.

### Getting Help

If you have questions or need support, please refer to the FAQ section or use the contact information provided.
